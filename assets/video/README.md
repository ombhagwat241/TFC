# Video Assets

Place your background video files here:

| File | Format | Notes |
|---|---|---|
| `trek-bg.mp4` | MP4 (H.264) | Primary — widest browser support |
| `trek-bg.webm` | WebM (VP9) | Optional fallback — smaller file size |

> **Tip:** Aim for 1920×1080, 10–30 seconds looping, under 15 MB for fast load.
> Free trekking/nature videos: [Pexels](https://www.pexels.com/search/videos/trekking/) · [Pixabay](https://pixabay.com/videos/search/hiking/)
